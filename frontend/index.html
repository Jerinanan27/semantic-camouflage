Semantic Camouflage

AI models can now read source code and flag security vulnerabilities — but they read the comments too. This demo shows how wrapping genuinely vulnerable C/C++ code in fake "all-clear" comments, without changing a single line of executable code, can fool the model into calling it safe. A multi-stage defense then strips the deceptive comments and restores the correct verdict.
